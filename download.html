Installation

Zum Installieren von aidGer laden Sie sich einfach die Datei unter dem Punkt "Programm (.jar)" herunter. AidGer braucht im Prinzip keinen eigenen Ordner, da alle Einstellungen im Benutzerverzeichnis abgespeichert werden (siehe Handbuch). Deshalb können Sie die Datei z.B. einfach auf Ihrem Desktop ablegen und von dort aus per Doppelklick starten. Vorraussetzung dafür ist, dass Sie die Java-Laufzeitumgebung von Oracle installiert haben. Diese können Sie auch nachträglich hier (externer Link) herunterladen.

Unter Mac OS X oder Linux können Sie aidGer auch per Kommandozeile starten, falls ein Doppelklick nicht zum gewünschten Ergebnis führt. Führen Sie dazu einfach folgendes aus: 

java -jar aidger-1.0.jar

Einrichtung

Beim ersten Start von aidGer begrüßt Sie der Ersteinrichtungsassistent, der mit Ihnen die wichtigsten Einstellungen vornimmt. Das ist zum einen Ihr Name, den Sie ein einfach in der Form <Vorname> <Nachname> eingeben.

Als nächstes folgen nun die Einstellungen zur Datenbankverbindung. Um aidGer nur auf Ihrem PC zu testen, belassen Sie einfach die bereits voreingestellten Werte und drücken solange auf "Weiter" bis der Assistent fertig ist.
