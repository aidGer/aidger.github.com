Systemvorraussetzungen

Unterstützt wird aidGer auf Windows XP, Vista und 7 sowie Server 2003 und 2008. Auf anderen Windowsversionen kann aidGer durchaus funktionieren, dies wird von uns jedoch nicht sichergestellt.

aidGer benötigt eine Java-Laufzeitumgebung mit mindestens Version 1.6 um ausgeführt werden zu könnep>n. Eine solche erhalten Sie zum Beispiel bei Oracle (hier (externer Link))

Als Datenbanken unterstützt aidGer sowohl MySQL als auch Derby. Derby is dabei im Lieferumfang enthalten und kann sofort eingesetzt werden. Um stattdessen jedoch eine MySQL-Datenbank zu benutzen, benötigen Sie MySQL 5.0 oder höher. MySQL ist die empfohlene Datenbank.

Anleitung

Eine Installation ist nicht nötig. Starten Sie aidGer durch einen Doppelklick auf die heruntergeladene Datei.
Beim ersten Start begrüßt Sie der Einrichtungsassistent und erlaubt Ihnen eine einfache Konfiguration der Datenbankverbindung und Ihres Namens.

Für mehr Informationen laden Sie sich das Handbuch herunter oder öffnen Sie es aus aidGer heraus über das Menü Hilfe -> Handbuch (rly?)

Systemvorraussetzungen

Unterstützt wird aidGer auf aktuellen Linux-Distributionen mit Linux 2.6 Kernel.

Zur Ausführung wird eine Java-Laufzeitumgebung mit Version 1.6 benötigt. Eine solche sollten Sie im Repository Ihrer Distribution finden. Unter Ubuntu/Debian z.B. entweder im Paket openjdk-6-jre für OpenJDK oder sun-java6-jre für die offizielle Sun/Oracle-Version.

Als Datenbanken unterstützt aidGer sowohl MySQL als auch Derby. Derby is dabei im Lieferumfang enthalten und kann sofort eingesetzt werden. Um stattdessen jedoch eine MySQL-Datenbank zu benutzen, benötigen Sie MySQL 5.0 oder höher. MySQL ist die empfohlene Datenbank.
